# MOEX Select Frontend

Пользовательский интерфейс сервиса **MOEX Select** от **Delta team**, реализованный на React, Vite и TypeScript.

> Информация не является индивидуальной инвестиционной рекомендацией. Подбор основан на выбранных пользователем параметрах и открытых рыночных данных MOEX ISS.

## Запуск

```powershell
npm.cmd install
npm.cmd run dev
```

Интерфейс откроется на `http://localhost:5173` и обращается к API на `http://localhost:8000/api`.

## Пользовательский сценарий

- Пользователь указывает цель, уровень риска, горизонт, сумму, опыт и классы инструментов.
- Интерфейс показывает карточки с доступными рыночными сведениями, понятными уровнями риска и ликвидности и причинами подбора.
- Для фьючерсов и опционов отображается сообщение о сложности инструмента.
- Кнопка `Открыть на MOEX` переводит на страницу инструмента.

Числовые внутренние оценки ранжирования в интерфейс не выводятся.
